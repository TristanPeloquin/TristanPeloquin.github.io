3D-Renderer

The first project I worked on, this 3D-renderer was written from scratch using Java/JavaFX and uses the Phong shading model in order to create the image you see above! It includes recursive reflections, specular highlights, shadows, and much more! Overall, it was great practice for applying my critical thinking skills to a real issue, and taught me a lot about structuring my code in a both readable and usable manner, not to mention great practice with front-end development and adapting to new frameworks.
